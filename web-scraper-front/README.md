# web-scraper-front

## Requirements

- node 8.11.x
- vue.js

## config

you can update baseApi in *./src/config.ts*

## Run

```
npm install
npm run lint
npm run serve
```

after success run, you can open http://127.0.0.1:8080/

click "Click here to EC site settings" to EC site settings page

## Build

you also can use `npm run build` to build app

```
npm run build
```
